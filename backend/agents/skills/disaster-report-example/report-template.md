# 报告结构模板

## 1. 执行摘要（200字内）
- 灾害类型和时间
- 主要影响区域
- 核心统计数据
- 关键建议

## 2. 灾情概览
### 2.1 基本信息
- **发生时间**：YYYY-MM-DD 至 YYYY-MM-DD
- **影响区域**：列出所有受影响的行政区
- **灾害类型**：洪涝/台风/暴雨

### 2.2 核心数据
- 受灾人口：XX万人
- 直接经济损失：XX亿元
- 农作物受灾面积：XX千公顷
- 倒塌房屋：XX间

## 3. 时空演化分析
### 3.1 时间演化
使用时间轴或折线图展示灾害发展过程：
- 降雨量变化趋势
- 水位变化曲线
- 受灾范围扩展时序

### 3.2 空间分布
使用地图或热力图展示：
- 受灾区域空间分布
- 损失程度的地理差异
- 重点关注区域标注

## 4. 影响分析
### 4.1 人员影响
- 受灾人口统计（按地区分类）
- 转移安置人数
- 人员伤亡情况（如有）

### 4.2 经济影响
- 直接经济损失分类统计
  - 农业损失
  - 工业损失
  - 基础设施损失
- 按地区的损失分布

### 4.3 设施影响
- 交通设施受损情况
- 水利设施受损情况
- 其他基础设施影响

## 5. 因果分析
### 5.1 成因分析
- 气象因素（降雨量、台风等）
- 地形地貌因素
- 人为因素（如有）

### 5.2 影响传播链
使用因果图或流程图展示：
- 初始事件 → 次级影响 → 连锁反应
- 例如：强降雨 → 水位上涨 → 堤坝溃堤 → 下游淹没

## 6. 应急响应与建议
### 6.1 已采取的措施
根据预案检索结果，列出已执行的应急响应：
- 应急响应等级
- 已实施的措施
- 投入的应急资源

### 6.2 后续建议
- 短期措施（灾后恢复）
- 中期措施（重建规划）
- 长期措施（防灾减灾）

## 7. 附录
- 数据来源说明
- 参考预案文档
- 相关图表汇总

---

## 格式要求

1. **标题层级**：使用 Markdown 标题（#, ##, ###）
2. **数据表格**：使用 Markdown 表格展示结构化数据
3. **可视化**：每个关键数据点都应配图表
4. **引用标注**：数据来源使用脚注或括号标注
