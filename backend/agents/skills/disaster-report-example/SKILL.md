# 灾害报告生成 Skill

## 概述

此 Skill 帮助你生成完整的灾害分析报告。报告应包含数据支撑的事实分析和可视化展示。

## 核心流程

### 1. 数据收集
使用 `query_knowledge_graph_with_nl` 查询灾害基本信息：
- 灾害发生时间和地点
- 受灾范围和影响区域
- 人员伤亡和经济损失

### 2. 时空分析
使用 `analyze_temporal_pattern` 分析灾害的时序演化规律。

### 3. 因果分析
使用 `find_causal_chain` 追踪灾害的成因和影响传播链条。

### 4. 应急建议
使用 `query_emergency_plan` 检索相关的应急响应措施和预案。

### 5. 数据可视化
使用 `generate_chart` 生成趋势图表展示关键数据。

## 报告结构模板

完整的报告结构和格式规范请参见 [report-template.md](report-template.md)

## 数据验证

在生成报告之前，建议使用验证脚本检查数据完整性：

```bash
python scripts/validate_data.py
```

该脚本会检查：
- 必需字段是否存在
- 数值范围是否合理
- 时间格式是否正确

## 注意事项

1. **数据来源**：所有数据必须来自工具查询结果，禁止编造
2. **缺失标注**：如果某个环节数据缺失，明确标注"数据不足"
3. **可视化优先**：优先使用图表展示趋势和对比
4. **引用预案**：应急建议需引用具体的预案条款
