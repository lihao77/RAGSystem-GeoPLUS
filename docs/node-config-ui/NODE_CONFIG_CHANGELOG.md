- ✅ 支持自动推断字段分组和UI格式
- ✅ 支持自定义UI元数据配置

- ✅ 支持6种智能控件类型

### 控件类型

支持的智能控件：
1. **文本输入框** - 普通字符串
2. **密码输入框** - 敏感信息（自动识别）
3. **多行文本框** - 长文本内容
4. **数字输入器** - 数值参数（支持范围和步长）
5. **开关按钮** - 布尔值
6. **下拉选择框** - 有限选项
7. **JSON编辑器** - 复杂对象/数组

### 配置分组

支持的分组类型：
- `default` - 基础配置
- `api` - API配置
- `database` - 数据库配置
- `model` - 模型配置
- `template` - 模板配置
- `processing` - 处理配置
- `metadata` - 元数据配置
- `output` - 输出设置
- `advanced` - 高级配置
